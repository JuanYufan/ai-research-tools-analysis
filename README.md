# AI用户研究工具竞品分析

此网站展示AI用户研究工具的竞品用户原声分析，包含以下8个维度：

1. 核心观点
2. 使用了什么AI用研工具
3. 原文
4. 信息来源网址
5. 用户评价日期
6. 产品名称
7. 融资节奏
8. 官网

## 部署到GitHub Pages

1. 访问 https://github.com/new 创建新仓库
2. 仓库名称设置为 `yourusername.github.io`
3. 上传此文件夹的所有内容到仓库
4. 访问 Settings -> Pages
5. Source 选择 "Deploy from a branch"
6. 保存设置

稍等几分钟，网站将在 https://yourusername.github.io 访问

## 部署到Netlify

1. 访问 https://netlify.com
2. 点击 "Get Started"
3. 拖拽整个文件夹到上传区域
4. 网站将立即上线

## 自动更新

要更新数据，只需替换index.html文件并重新部署。
